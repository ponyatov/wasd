# Что такое [[WASD]] ?

- общеизвестное сочетание кнопок на игровой клавиатуре
- **виртуальная машина для разработки компактных программ**
- [[Форт]]-архитектура:
	- `W`ords = Vocabulary
	- `A`sync message queue
	-  return `S`tack
	-  `D`ata stack
